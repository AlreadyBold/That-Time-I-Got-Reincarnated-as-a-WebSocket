## Websocket Toy Project

Websocket을 사용한 채팅방 구현 프로젝트

## 제작 기한

2023-06 -

## 개발 환경

- Java 17, Jar
- Spring Boot 3.2.12
- Mysql 
- Mongo DB
- Thymeleaf ( 안쓸수도 있음 )

## 기능
- 회원
    - 회원가입
    - 회원정보수정
    - 로그인
    - 로그아웃
    - 회원정보조회
    - 회원탈퇴
    - 채팅 알림 기능
- 채팅
    - 채팅기능
    - 채팅저장
    - 채팅삭제
    - 채팅요청
